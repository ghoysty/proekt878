## Описание программы
*"Training Planner (План тренировок) — это консольное приложение для планирования тренировок. Позволяет
добавлять записи о планировании тренировок, фильтровать их по дате и добавляет графики тренировок."*
---
## Требования для запуска
Что нужно установить на компьютер:
- Python 3.10 или выше
- Библиотеки: `pip install matplotlib requests`
